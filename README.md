# Zenborg

### Make your garden flourish.

> A habit cultivation tool to plan and tend your daily moments of intention.

**[Try it out](https://zenborg.vercel.app/)** | [Download the desktop app](https://github.com/Thopiax/zenborg/releases/latest)

## What is Zenborg?

Zenborg is a simple tool to make your intentions explicit, tangible and adaptable throughout the day.

It looks like a Kanban board to drag and drop moments of intention throughout the day.

1. Define **Areas** of flourishing (e.g. fitness, mindfulness, hobbies, friends, family...)
2. Plan **Moments** to cultivate these areas
3. Commit these moments to the **Timeline**
4. Stay on track with the **Compass**
5. Review past moments and reflect

*"Where will I place my consciousness today?"*

## Liberating Limits

We tend to see limits as a problem. If we are limited, we aren't free. If we aren't free, well, that's just plain bad.

Zenborg embraces constraints as liberation:

- **3 words** to a moment
- **3 moments** per phase
- **3 phases** per day
- **3 days** in the timeline

## What this is NOT

It looks like just another productivity tool, but...

- **Not a task manager**: no tasks, no checklists, no "done", no projects, no goals, no outcomes
- **Not a habit tracker**: no streaks, no points, no badges — and definitely no leaderboards
- **Not a calendar**: no durations, no "time blocking", no reminders, no notifications
- **Not even an app**: no accounts, no servers, no analytics, no ads

All of these tools can be useful in their own right. Zenborg explores a mindful alternative to these conventional paradigms.

## Quick Start

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Open http://localhost:3000
```

## Tech Stack

- **Next.js 15** — React framework with App Router
- **TypeScript** — Type-safe development
- **Tailwind CSS 4** — Utility-first styling
- **Legend State** — Reactive local-first state management
- **Tauri 2** — Native desktop app (macOS)
- **Vitest + Playwright** — Unit and E2E testing
